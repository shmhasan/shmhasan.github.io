# Ummah Platform — Static Prototype

A multi-page static HTML prototype for a Community Support & Fund Management Platform. Built with vanilla HTML/CSS/JS and designed for GitHub Pages.

## Quick Start
- Open `index.html` in your browser, or
- Serve the folder with any static server.

## Project Structure
- `index.html` and public pages live at the root.
- `app/` contains the logged-in dashboard pages.
- `assets/css/styles.css` is the shared design system.
- `assets/js/app.js` handles shared interactions and UI helpers.
- `assets/js/mock-data.js` provides mock data (no backend).

## Role Switching
Use the role switcher in the topbar to preview Admin, Treasurer, Auditor, and Member experiences.

## Notes
- All links are relative and GitHub Pages compatible.
- No build tools or frameworks required.
- This is a prototype (frontend-only), not a production system.
